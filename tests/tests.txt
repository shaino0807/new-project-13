# Tests

## Test 1 - Homepage waits for user input [sanity]
Viewport: desktop
Description: Verifies the public site no longer auto-loads a default stock before the visitor enters a code.

Steps:
1. Navigate to the app without a `code` query parameter.
2. Wait for the page to fully load.

Expected: The hero shows 台股技術分析 Agent and a 待輸入 / 待載入 state. It must not show 2330 台積電, a default close price, or a default pullback probability before user input.

## Test 2 - Visitor can analyze 2344
Viewport: desktop
Description: Verifies a shared visitor can enter a stock name or code and receive backend quote analysis on the same page.

Steps:
1. Navigate to the app.
2. Type 2344 in the stock code or name field.
3. Click the analysis button.
4. Wait for the heading and metrics to update.
5. Change monthly budget to 5000.

Expected: The page shows 2344 with stock metrics, technical sections, value radar, allocation guide, fundamentals, news, charts, and source/status text. Budget-dependent sections update without reloading the page.

## Test 10 - Dino swing workflow is customer-facing and skill-aligned
Viewport: desktop and mobile
Description: Verifies Dino focuses on usable customer output rather than internal calculation steps.

Steps:
1. Navigate to the app.
2. Click Dino K棒判斷策略.
3. Wait for macro/news, 1-3 month screener, thematic tables, and source panel to load.
4. Select at least one candidate stock or enter 2330, 2454 manually.
5. Click 進入深度流程.
6. Wait for the deep workflow result.
7. Return to 股票深度頁 and run 2330.

Expected: The internal workflow explainer and 重新產生候選 button are not visible to normal users. The macro/news block explains article-specific key points, reasons, market impact, screening use, source, horizon, and whether article content or RSS summary was read. The screener table shows detailed 1-3 month setup descriptions, main evidence, research invalidation line with conditions, and priority. The thematic block shows five tables with readable business/theme notes and clear units for institutional net buy. The source panel shows full-market universe count plus the scored seed count. Deep analysis requires selected or manually entered stock codes and does not break the original stock analysis page.
