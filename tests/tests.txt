# Tests

## Test 1 - Homepage loads and default analysis appears [sanity]
Viewport: desktop
Description: Verifies the public site opens to the stock analysis Agent and loads a default stock without showing a blank page.

Steps:
1. Navigate to the app.
2. Wait for the page to fully load.

Expected: The page shows the title "台股技術分析 Agent" or a loaded stock heading, plus metric cards and the status text mentioning a data source or latest data request.
