## Test 10 - Value radar and ranking features load
Viewport: desktop
Description: Verifies the Investing.com-inspired value radar is integrated into the existing page, not a separate site.

Steps:
1. Navigate to the app.
2. Wait for the default stock analysis and supplemental data to finish loading.
3. Observe 小資價值雷達.
4. Click 最被高估, 觀察池最活躍, 現金流最佳, 增長最佳, and 小資友善.

6. Enter "0050 40%, 2330 40%, 現金 20%" in 持股健康檢查 and click 檢查配置.

Expected: The same page shows 小資友善分數, 低估分數, 高估分數, 公允價值, 現金流評分, 增長評分, 市場信心, 分析師評級 status, and 追高風險警示. The ranking table updates without leaving the page. Additional expected: The value radar shows a data transparency panel with quote date, coverage rate, dataset dates, cache/stale status, and missing data items. The ranking note identifies the maintainable Taiwan universe version, scored/requested count, and states that it is not a real-time full-market database.

## Test 11 - Client workbench stays separate from stock depth page
Viewport: desktop and mobile
Description: Verifies paid-client workflow features live in a separate tab while the existing stock analysis page remains available.

Steps:
1. Navigate to the app.
2. Confirm the default stock depth page still shows the existing quote, charts, fundamentals, value radar, strategy, and Agent sections.
3. Click 客戶工作台.
4. Wait for the workbench load state to resolve.
5. Inspect 評級標準與排行榜範圍.
6. Click ranking scope filters: 觀察池, 個股, ETF.
7. Click ranking metric filters: 低估, 高估, 現金流, 成長, 小資, ETF.
8. In 我的持股風險, enter 0050 40%, 2330 30%, 2454 10%, cash 20% and click 分析持股.
9. Click a stock from 今日值得觀察清單, 持股分數, or 價值評估清單.
10. Return to 客戶工作台 and inspect 回測實驗室 v1 and AI 理財財務長.

Expected: 客戶工作台 is a separate page view, not appended into the existing stock depth flow. It shows 評級標準與排行榜範圍, 市場狀態, 我的持股風險, 持股分數, 今日值得觀察清單, 值得觀察清單, 追高風險清單, 價值評估清單, scope filters, metric filters, 回測實驗室 v1, and AI 理財財務長. 評級標準 lists the stock weights 30/20/15/15/10/10 and ETF weights 25/25/15/15/10/10, while clearly saying unavailable inputs such as EV/EBITDA, same-industry comparison, ROE, margins, and dividend stability are only used when available. 排行榜範圍 explains 觀察池排行, 可評級台股池, and ETF 獨立榜 without claiming full-market coverage. Portfolio holdings with available universe scores show grade, total score, chase-risk score, and data coverage; holdings without a score are marked 資料不足 and are not assigned invented values. The stock depth page keeps the original analysis flow and shows a top-level 價值雷達 grade/total/coverage badge beside the stock result plus 價值雷達總分 with A/B/C/D/X and expandable weighted components. AI 理財財務長 shows stage/wait/avoid style guidance, single-entry cap, max holding cap, worst-case loss estimate, and a stop-tracking assumption without direct buy/sell instructions. All labels remain readable without overlap or replacement-character mojibake.
